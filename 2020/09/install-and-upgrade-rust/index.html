安装 rustup

使用 rustup 安装 rust ，安装 rustup:

1

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

在 Rust 开发环境中，所有工具都安装在 ~/.cargo/bin 目录中，您可以在这里找到包括 rustc、cargo 和 rustup 在内的 Rust 工具链。

查看 rust 版本

1

rustc --version

卸载

1

rustup self uninstall

升级

1

rustup update
