Mongoose介绍

Mongoose 是 MongoDB 的一个对象模型工具，是基于node-mongodb-native开发的MongoDB 的 Node.js 驱动，可以在异步的环境下执行。同时它也是针对 MongoDB 操作的一个对象模型库，封装了 MongoDB 对文档的一些增删改查等常用方法，让 Node.js 操作 MongoDB 数据库变得更加容易。
