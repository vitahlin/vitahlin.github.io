__defineGetter__介绍

发表于 2017-08-03 | 分类于 JavaScript |

概述

__defineGetter__ 方法可以将一个函数绑定在当前对象的指定属性上，当那个属性的值被读取时，你所绑定的函数就会被调用。

语法

  • prop 一个字符串，表示指定的属性名
  • func 一个函数，当 prop 属性的值被读取时自动被调用。
