macOS上Node.js环境的安装

发表于 2016-11-29 | 分类于 Node.js |

系统版本macOS 10.12.1，默认安装Homebrew

安装nvm

通过brew安装，执行命令brew install nvm进行安装。
nvm是用来管理Node版本的工具。

安装完后还需要进行设置，创建nvm的工作目录，用来保存下载的各个node版本。

安装Node

用命令nvm ls-remote查看可安装的Node版本，有时候因为网络问题可能只会显示iojs-x.x.x的版本，多尝试几次命令即可。

查看列表后，可以安装想要安装的版本，例如：nvm install v5.12.0，安装完成后，可以用命令node -v查看Node.js版本。
