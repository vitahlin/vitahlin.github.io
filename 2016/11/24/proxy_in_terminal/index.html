Mac终端翻墙

发表于 2016-11-24 | 分类于 Linux&Unix |

系统版本macOS 10.12.1，需要shadowsocks支持，默认系统已安装Homebrew

安装proxychains-ng

设置

编辑配置文件vim /usr/local/etc/proxychains.conf在 [ProxyList] 下面（也就是末尾）加入代理类型，代理地址和端口，例如使用 TOR 代理，注释掉原来的代理并添加socks5 127.0.0.1 1080

使用

安装时在命令前加上proxychains4，如：

如果使用zsh可设置.zshrc文件简化命令：

保存后运行source ~/.zshrc。
这样的话命令proxychains4 brew install XXX可简化为pc brew install XXX
