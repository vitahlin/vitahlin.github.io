发表于 2016-11-28 | 分类于 Vim |

系统版本macOS 10.12.1，默认安装Homebrew

安装

通过brew安装，执行命令brew install macvim进行安装。
brew的用法可以查看介绍：[链接地址]

.vimrc设置

.vimrc文件位置
